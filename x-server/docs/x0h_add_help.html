X0hdb Submission Help

Parameters Space Group (structure symmetry group), Density (material density in g/cm3) and Z (formula units per cell) are used for verifying structure consistency.

First of all, the selected Space Group transformations are applied to the supplied coordinates of atoms. If you believe that you provided all coordinates of atoms in the unit cell, the transformations should not add any new positions. You can also provide only initial positions and then the space group transformations will give you all positions. Let us take GaAs crystal as and example. It contains four Ga atoms and four As atoms in the unit cell. The space group of GaAs is 216 F-43m. If you specify all coordinates: 

Ga
 0.00,   0.00,   0.00
 0.00,   0.50,   0.50
 0.50,   0.00,   0.50
 0.50,   0.50,   0.00
As
 0.25,   0.25,   0.25
 0.25,   0.75,   0.75
 0.75,   0.25,   0.75
 0.75,   0.75,   0.25

then the space group transformation will add nothing. However, if you specify only a partial set, then the 216 F-43m space group transformations will produce the above four sites per atom. If after processing the program returns more (or less) coordinates than you expect, then either the initial coordinates or the space group are incorrect. Many space groups contain multiple settings. Make sure that you pick correct settings matching the coordinates you provide.

Parameter Density is used for the second structure check. Once all coordinates are found and the lattice parameters are supplied, the program can calculate the material density, which is checked against the supplied one for consistency. The check is considered passed in the difference is below 10%.

Parameter Z is used for the final check. Z is the number of chemical formula instances (units) contained in the unit cell. For example, for GaAs crystal, Z=4 because the unit cell contains 4 Ga atoms and 4 As atoms; for Silicon crystal, Z=8 because the unit cell of Silicon crystal contains 8 Si atoms. After applying the space group transformations, the program caclulates the number of sites for each atom and then divides it by Z. If everything is corret, it should give the checmical formula (e.g. Ga1As1 or Si1), but if not, then the number of coordinates for some atom or atoms s is incorrect, which means either the initial positions before applying the space group transformations are incorrect or incorrect space group was selected.

Please note that if occupations of atoms are not equal one (i.e. if atoms occupy their cells with the probability of less than 100%) the Z-check may be tricky and therefore one may skip it by not providing the expected Z value.
